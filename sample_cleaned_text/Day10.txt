Random Access
•
In random-access or contention no station is
superior to another station, and none is assigned
control over another.
•
At each instance, a station that has data to send uses
a procedure defined by the protocol to decide on
whether to send.
•
This decision depends on the state of the medium
(idle or busy).

ALOHA
•
It was designed for a radio (wireless) LAN, but it can
be used on any shared medium.
•
It is obvious that there are potential collisions in this
arrangement. The medium is shared between the
stations. When a station sends data, another station
may attempt to do so at the same time. The data
from the two stations collide and become garbled.

Example
•
A pure ALOHA network transmits 200-bit frames
on a shared channel of 200 kbps. What is the
requirement to make this frame collision-free?
•
Solution: Average frame transmission time T is 200
fr
bits/200 kbps or 1 ms. The vulnerable time is 2 × 1
ms = 2 ms. This means no station should send later
than 1 ms before this station starts transmission and
no station should start sending during the period (1
ms) that this station is sending.

Carrier sense multiple access
(CSMA)
•
To minimize the chance of collision and, therefore,
increase the performance, the CSMA method was
developed.
•
CSMA requires that each station first listen to the
medium (or check the state of the medium) before
sending. In other words, CSMA is based on the
principle “sense before transmit” or “listen before
talk.”

CSMA/CD
•
The CSMA method does not specify the procedure
following a collision. Carrier sense multiple access
with collision detection (CSMA/CD) augments the
algorithm to handle the collision.
•
In this method, a station monitors the medium after
it sends a frame to see if the transmission was
successful. If so, the station is finished. If, however,
there is a collision, the frame is sent again.

Example
•
A network using CSMA/CD has a bandwidth of 10
Mbps. If the maximum propagation time is 25.6 μs,
what is the minimum size of the frame?
•
Solution: The minimum frame transmission time is
T = 2 × Tp = 512.2 μs. This means, in the worst
fr
case, a station needs to transmit for a period of 512.2
μs to detect the collision. The minimum size of the
frame is 10 Mbps × 512.2 μs = 512 bits or 64 bytes.
